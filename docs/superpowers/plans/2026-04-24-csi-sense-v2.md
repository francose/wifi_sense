# CSI Sense v2 — Full Feature Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Transform the single-file CSI sensing firmware into a multi-page, touch-driven sensing platform with channel hopping, multi-AP tracking, occupancy counting, motion direction, IMU fusion, and alert mode on the Waveshare ESP32-S3-Touch-LCD-1.69.

**Architecture:** Split the monolithic `main.c` into focused modules: `lcd.c` (display primitives + font), `touch.c` (CST816T I2C driver), `imu.c` (QMI8658 driver), `csi_engine.c` (CSI processing + classification), `ap_tracker.c` (multi-AP state), `ui_pages.c` (page rendering), `channel_hop.c` (channel cycling timer). A `ui_state` struct holds current page index and is advanced by touch gestures. Each module has a clear header with its public API.

**Tech Stack:** ESP-IDF v5.2, C, SPI (ST7789V2 LCD), I2C (CST816T touch + QMI8658 IMU), FreeRTOS tasks

**Hardware:** Waveshare ESP32-S3-Touch-LCD-1.69 — LCD pins: MOSI=7, CLK=6, CS=5, DC=4, RST=8, BL=15. Touch: SDA=11, SCL=10, RST=13, INT=14, addr=0x15. IMU: same I2C bus (0x6B).

- [ ] **Step 2: Create touch.c** — I2C master init on GPIO 10 (SCL) / 11 (SDA). Reset touch via GPIO 13. Configure interrupt on GPIO 14 (optional, can poll). CST816T register map: reg 0x01 = gesture ID (0x00=none, 0x01=swipe up, 0x02=swipe down, 0x03=swipe left, 0x04=swipe right, 0x05=single tap). Read gesture register, map to enum, return.

```c
#include "touch.h"
#include "driver/i2c.h"
#include "driver/gpio.h"
#include "freertos/FreeRTOS.h"
#include "freertos/task.h"

#define TOUCH_SDA   GPIO_NUM_11
#define TOUCH_SCL   GPIO_NUM_10
#define TOUCH_RST   GPIO_NUM_13
#define TOUCH_INT   GPIO_NUM_14
#define TOUCH_ADDR  0x15
#define I2C_PORT    I2C_NUM_0

void touch_init(void) {
    // Reset
    gpio_config_t io = { .pin_bit_mask = (1ULL << TOUCH_RST), .mode = GPIO_MODE_OUTPUT };
    gpio_config(&io);
    gpio_set_level(TOUCH_RST, 0);
    vTaskDelay(pdMS_TO_TICKS(50));
    gpio_set_level(TOUCH_RST, 1);
    vTaskDelay(pdMS_TO_TICKS(100));

    // I2C init
    i2c_config_t conf = {
        .mode = I2C_MODE_MASTER,
        .sda_io_num = TOUCH_SDA,
        .scl_io_num = TOUCH_SCL,
        .sda_pullup_en = GPIO_PULLUP_ENABLE,
        .scl_pullup_en = GPIO_PULLUP_ENABLE,
        .master.clk_speed = 400000,
    };
    i2c_param_config(I2C_PORT, &conf);
    i2c_driver_install(I2C_PORT, I2C_MODE_MASTER, 0, 0, 0);
}

touch_gesture_t touch_read(void) {
    uint8_t reg = 0x01;
    uint8_t gesture = 0;
    i2c_master_write_read_device(I2C_PORT, TOUCH_ADDR, &reg, 1, &gesture, 1, pdMS_TO_TICKS(50));
    switch (gesture) {
        case 0x05: return GESTURE_TAP;
        case 0x03: return GESTURE_SWIPE_LEFT;
        case 0x04: return GESTURE_SWIPE_RIGHT;
        case 0x01: return GESTURE_SWIPE_UP;
        case 0x02: return GESTURE_SWIPE_DOWN;
        default:   return GESTURE_NONE;
    }
}
```

```c
static void ui_task(void *arg) {
    lcd_rect(0, 0, LCD_W, LCD_H, C_BLK);
    while (1) {
        touch_gesture_t g = touch_read();
        if (g == GESTURE_SWIPE_LEFT) {
            g_ui.current_page = (g_ui.current_page + 1) % UI_NUM_PAGES;
            lcd_rect(0, 0, LCD_W, LCD_H, C_BLK); // clear on page change
        } else if (g == GESTURE_SWIPE_RIGHT) {
            g_ui.current_page = (g_ui.current_page + UI_NUM_PAGES - 1) % UI_NUM_PAGES;
            lcd_rect(0, 0, LCD_W, LCD_H, C_BLK);

            g_ui.alert_enabled = !g_ui.alert_enabled;
        }

        switch (g_ui.current_page) {
            case 0: ui_page_overview(); break;
            case 1: ui_page_waveforms(); break;
            case 2: ui_page_ap_list(); break;
            case 3: ui_page_imu(); break;
            case 4: ui_page_alert(); break;

            lcd_rect(0, 35, 3, 225, C_RED);       // left
```

- [ ] **Step 2: Implement occupancy estimation in csi_engine.c**

After classification in `process_csi()`, count how many subcarriers have variance > 2x the mean. More active subcarriers = more people disturbing more signal paths.

```c
int high_var_count = 0;
float mean_var = total_var / nsc;
for (int i = 0; i < nsc; i++) {
    if (g_csi.sc_variance[i] > mean_var * 2.0f) high_var_count++;
}
if (high_var_count < 3) g_csi.occupancy_estimate = 0;
else if (high_var_count < 8) g_csi.occupancy_estimate = 1;
else if (high_var_count < 16) g_csi.occupancy_estimate = 2;
else g_csi.occupancy_estimate = 3;
```

- [ ] **Step 3: Implement motion direction**

Split 64 subcarriers into 3 groups (0-15, 16-47, 48-63). Track variance per group. If low-group variance is rising while high-group is falling, motion is toward lower frequencies (one direction). Reverse = other direction.

```c
// Compute group variances
float gv[3] = {0};
for (int i = 0;  i < 16; i++) gv[0] += g_csi.sc_variance[i];
for (int i = 16; i < 48; i++) gv[1] += g_csi.sc_variance[i];
for (int i = 48; i < 64; i++) gv[2] += g_csi.sc_variance[i];
gv[0] /= 16; gv[1] /= 32; gv[2] /= 16;

static float prev_gv[3] = {0};
float d0 = gv[0] - prev_gv[0];
float d2 = gv[2] - prev_gv[2];
memcpy(prev_gv, gv, sizeof(gv));
memcpy(g_csi.sc_group_var, gv, sizeof(gv));

if (fabsf(d0) < 0.1f && fabsf(d2) < 0.1f) {
    g_csi.motion_direction = "NONE";
} else if (d0 > d2 + 0.2f) {
    g_csi.motion_direction = "LEFT";
} else if (d2 > d0 + 0.2f) {
    g_csi.motion_direction = "RIGHT";
} else if (d0 > 0 && d2 > 0) {
    g_csi.motion_direction = "TOWARD";
} else {
    g_csi.motion_direction = "AWAY";
}
```

10-segment horizontal bar. Map RSSI -90 to -30 dBm across 10 segments. Color: segments 1-3 = red (weak), 4-6 = yellow (ok), 7-10 = green (strong).

```c
void ui_draw_signal_meter(int x, int y, int rssi) {
    int segments = (rssi + 90) / 6;  // -90=0, -30=10
    if (segments < 0) segments = 0;
    if (segments > 10) segments = 10;
    for (int i = 0; i < 10; i++) {
        uint16_t col;
        if (i < 3) col = C_RED;
        else if (i < 6) col = C_YEL;
        else col = C_GRN;
        if (i >= segments) col = C_DKG;
        lcd_rect(x + i * 12, y, 10, 8, col);
    }
}
```
